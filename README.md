# 천아인 렌탈 관리자 알림 최종본

## 실행
```bash
npm install
npm run dev
```

## Vercel 배포
- Build Command: npm run build
- Output Directory: dist

## 관리자 코드
CHEONAIN2026

## 포함 기능
- 장바구니 가격 미노출
- 장바구니 전체 초기화 / 개별 삭제
- 관리자 새 견적 알림 배지
- 관리자 메뉴 새 견적 알림 박스
- 관리자 견적 표
- 엑셀 다운로드
- 공급가액 / 부가세 10% / 최종 금액 계산
- 관리자 물품 추가 / 가격 수정
- 사진 추가 / 삭제 구조
