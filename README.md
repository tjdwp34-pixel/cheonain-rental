# 천아인 렌탈 완성본

## 실행
```bash
npm install
npm run dev
```

## Vercel 배포
- Framework: Vite
- Build Command: npm run build
- Output Directory: dist

## 관리자 코드
CHEONAIN2026

## 포함 기능
- 관리자 모드 / 견적 확인
- 관리자 전용 견적 금액 자동 산출 + 부가세 10%
- 관리자 물품 관리에서 현재 단가 표시/수정
- 견적 문의 및 관리자 물품 관리 카드 사이즈 통일
- Firebase 사진 업로드/삭제 코드
- 미리보기 환경 Firebase 예외 처리
