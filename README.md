# 천아인 렌탈 - 분리 최적화 버전

## 실행 방법

```bash
npm install
npm run dev
```

## Vercel 배포 설정

- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist

## 관리자 코드

```txt
CHEONAIN2026
```

## 포함 기능

- 파일 분리 구조
- Firebase Storage 사진 업로드/삭제
- Firebase Firestore 사진 URL 저장
- Firebase Firestore 물품 추가/삭제 저장
- 관리자 모드: 사이트 물품 관리 / 견적 확인
- 견적 문의
- 이전 견적 검색
- 수량칸 잘림 방지
- 견적 품목 5칸 고정 그리드
